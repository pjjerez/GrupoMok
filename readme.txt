Para ejecutar las API siga los siguientes Pasos

1. Descomprima el archivo GrupoMok en una carpeta
2. Cambie el nombre del server en el archivo ApiGrupoMok\DatosGrupoMok\ParametrosGrupoMok.resx, "MEMO" por el nombre de su server, linea 121
3. Abra las soluciones con Visual Studio 2022
4. Ejecute el script GrupoMok\BaseDatosSQlServer\GrupoMok.sql o attache la base de datos adjunta 
5. Compile y ejecute las soluciones, estas API se puede probar con swagger y/o Postman
